Videopixie is a video editing marketplace marketplace. They use AngularFire to power real-time chat, so that buyers and editors can work together in real-time.

NoogaJS is an open source and JavaScript focused community, who built their website using AngularFire. [Source Code]

TODOMVC is a AngularFire port of the popular TODO app that helps you pick the right MV* framework. [Source Code]

Bring the Salsa is a way to make hosting parties easier by creating shared food and drink lists, built with AngularFire. [Source Code]

Firetube is a simple example of using AngularFire to secure a YouTube-like comment system. [Source Code]

hack.g0v is a way to organize dynamic documents during hackathons. It uses AngularFire to synchronize the document list in real-time. [Source Code]
